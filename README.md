## ✏️ 서비스 소개

플로밋은 흩어진 아이디어와 회의 내용, 자료를 노드 기반으로 연결해
</br>
프로젝트 기획 흐름을 시각화하는 AI 기반 기획 협업 플랫폼입니다.

사용자는 복잡한 기획 구조를 한눈에 확인하고,
</br>
AI 회의 요약과 AI 에이전트를 통해 핵심 내용을 빠르게 정리할 수 있습니다.

또한 팀원들과 기획 내용을 공유 및 관리하며,
</br>
프로젝트의 흐름과 맥락을 놓치지 않고 협업할 수 있도록 돕습니다.

</br>
